## Система бронирования компьютеров

Требования:
- Пользователь может выбрать дату, затем временной диапазон с шагом в 1ч
- Видит информацию о компьютерах, какие свободны в это время, а какие заняты
- Может выбрать комп (или несколько) и увидеть сумму к оплате
- Можно нажать кнопку забронировать, после чего дается 15 мин на оплату
- Оплатить можно с депозита. Если на депозите недостаточно средств, то надо его пополнить
- Можно посмотреть историю бронирований

### Сущности
User
- id
- login
- email
- password_hash
- username - строка ФИО
- deposit - депозит в рублях

Computer
- id
- name
- price - цена за час
- specs - CPU, GPU, монитор и др. инфа

Данные компов уже будут в системе
Доступен или нет комп можно увидеть через его бронирования

Booking
- id
- user_id
- code - уникальный код бронирования, 6 чисел
- computers - компьютеры в брони 
- start_time - начало бронирования, формат: 2025-01-01 12:00:00
- end_time - конец бронирования, формат: 2025-01-01 12:00:00
- status - confirmed - бронь подтверждена, canceled - бронь отменена
- totalPrice

Если пользователь отменил бронь, статус меняется на cancelled и комп доступен для бронирований
После успешного бронирования статус переход в confirmed, устанавливается интервал бронирования
